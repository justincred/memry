System Architecture
┌─────────────────────────────────────────────────────────┐
│  HintModeProvider (React Context)                       │
│  ┌───────────────────────────────────────────────────┐  │
│  │ State: isActive, hints[], typedChars, matchedHint │  │
│  │ Actions: activate(), deactivate(), typeChar()     │  │
│  └───────────────────────────────────────────────────┘  │
│                          │                              │
│            ┌─────────────┼─────────────┐                │
│            ▼             ▼             ▼                │
│  ┌──────────────┐ ┌───────────┐ ┌──────────────────┐   │
│  │ useHintMode  │ │ HintOverlay│ │ useHintActivation│   │
│  │ (hook)       │ │ (portal)  │ │ (hook)           │   │
│  │              │ │           │ │                  │   │
│  │ • scanDOM()  │ │ • render  │ │ • F key listener │   │
│  │ • assignHints│ │   badges  │ │ • Alt+F listener │   │
│  │ • matchInput │ │ • position│ │ • Esc to cancel  │   │
│  │ • clickTarget│ │   calc    │ │ • Esc in editor  │   │
│  └──────────────┘ └───────────┘ │   = blur editor  │   │
│                                  └──────────────────┘   │
│                          │                              │
│                          ▼                              │
│  ┌───────────────────────────────────────────────────┐  │
│  │ labelAssigner.ts (pure function)                   │  │
│  │                                                   │  │
│  │ Input: HTMLElement[]                              │  │
│  │ Output: { element, label, rect }[]                │  │
│  │                                                   │  │
│  │ Algorithm:                                        │  │
│  │ 1. Extract text from each element                 │  │
│  │ 2. Try mnemonic (first letter of text)            │  │
│  │ 3. Conflicts → try second letter                  │  │
│  │ 4. Still conflicts → two-char code (AJ, AK...)   │  │
│  │ 5. Return sorted by DOM position                  │  │
│  └───────────────────────────────────────────────────┘  │
│                                                         │
│  ┌───────────────────────────────────────────────────┐  │
│  │ domScanner.ts (pure function)                      │  │
│  │                                                   │  │
│  │ Selector: button, a, [role=button], [role=tab],   │  │
│  │   [role=treeitem], [role=menuitem], [tabindex],   │  │
│  │   [data-hint]                                     │  │
│  │                                                   │  │
│  │ Filters:                                          │  │
│  │ • Must be visible (offsetParent !== null)          │  │
│  │ • Must be in viewport                             │  │
│  │ • Not disabled                                    │  │
│  │ • Not inside hint overlay itself                  │  │
│  │ • Min size: 8x8px                                │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘

File Structure

src/renderer/src/
├── contexts/
│   └── hint-mode/
│       ├── context.tsx          ← HintModeProvider + React context
│       └── types.ts             ← Hint, HintModeState interfaces
├── hooks/
│   ├── use-hint-mode.ts         ← DOM scan + label assign + match logic
│   └── use-hint-activation.ts   ← F / Alt+F / Esc key listeners
├── components/
│   └── hint-overlay/
│       ├── hint-overlay.tsx      ← Portal with positioned badges
│       └── hint-badge.tsx        ← Individual badge component
└── lib/
    ├── dom-scanner.ts            ← querySelectorAll + visibility filters
    └── label-assigner.ts         ← mnemonic-first label algorithm
