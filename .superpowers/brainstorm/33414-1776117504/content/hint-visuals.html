Interaction Flow
1. Activation
Press F (or Alt+F from editor)
→ Badges appear on all clickable elements
→ Status bar shows "HINT MODE" indicator
→ All other shortcuts suspended
2. Narrowing
Type first character
→ Non-matching hints fade out (opacity 0.3)
→ Matched char dims in remaining badges
→ If unique match → go to step 3
3. Execution
Unique match found
→ Element receives programmatic .click()
→ Hint mode deactivates
→ Focus moves to clicked element
4. Cancellation
Press Esc at any time
→ All badges removed
→ Normal mode restored
→ No element clicked
Backspace
Undo last typed character
→ Re-expand visible hints
→ Empty typed chars? Back to full hint display
