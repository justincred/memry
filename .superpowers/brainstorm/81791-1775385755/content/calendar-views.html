Visual Language Summary

Calendar Events
• Solid left-border (calendar color)
• Filled background tint
• Time range shown
• Source label (Work, Google, etc.)
Tasks (due date)
• Dashed border (indigo)
• Transparent/subtle background
• Checkbox (☐) prefix
• "due today" / "due Fri" label
