3. Key Behaviors

Edge cases and interactions.

⌨️ Un-indent (Shift+Tab)

Subtask promotes to standalone task block. parentId → null. Renders at same level as parent.

🔗 Inheritance

Subtask auto-inherits projectId from parent. Priority defaults to none (user sets if needed).

🚫 Depth Limit

Only 1 level. Tab under a subtask row is a no-op — prevents infinite nesting.

📝 Markdown Format

Serialized as indented checkbox:
  - [ ] text {task:id}
Same {task:id} suffix pattern.

↗️ Navigate Arrow

Each subtask row has the same ↗ button. Opens subtask in Tasks panel with full detail view.

🔄 Sync

Subtask is a regular Task entity. Syncs via existing SyncItemHandler. Field clocks work as-is.
